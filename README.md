# Livia Melo

Landing page created for Livia Melo.

# Details

The library is built with:

- HTML5
- CSS 3
- [Tailwind CSS](https://tailwindcss.com/)

## Get Started

Install all the dependencies

```shell
npm run install
```

Start the project

```shell
npm run dev
```

## Build

Build Component Library

```shell
npm run build
```
